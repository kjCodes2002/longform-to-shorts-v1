# longform-to-shorts-v1

longform-to-shorts-v1 is a beta project that experiments with breaking long videos or podcasts into short, high-value clips.  
The goal is to automatically identify important moments and generate short-form friendly hooks and captions.

This repository is under active development and is primarily used for exploration and prototyping.
